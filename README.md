# TaianCGLIB2B-Frontend

泰安公責 B2B-前端

# Setting

configs loaded in following order in dev

```
.env.development
.env.gcp
.env.uat
.env.production
```

# Run Dev

```sh
npm run dev
```

# Run Build

```sh
#gcp
npm run build:gcp
# uat
npm run build:uat
# 正式環境
npm run build
```
